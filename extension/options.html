Changing Ports

Port configuration lives in config.json inside the extension directory:

  1. Edit code/extension/config.json with new port values
  2. Update docker-compose.yml port mappings to match
  3. Run docker compose up -d to restart containers
  4. Reload the extension in chrome://extensions

Most users won't need to change the defaults (backend: 8066, frontend: 5176).
